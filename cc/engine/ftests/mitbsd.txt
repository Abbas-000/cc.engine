Open Source Licenses
--------------------

Creative Commons provides license deeds for some open source licenses.
These deeds summarize the license and contain basic metadata about the
license.

  >>> browser = Browser()
  >>> browser.raiseHttpErrors = True
 
  >>> browser.open('http://localhost:8080/licenses/MIT')
  >>> browser.url
  'http://localhost:8080/licenses/MIT/'
  >>> 'MIT' in browser.contents
  True

  >>> browser.open('http://localhost:8080/licenses/BSD/')
  >>> 'BSD' in browser.contents
  True
  >>> 'No Endorsement' in browser.contents
  True
